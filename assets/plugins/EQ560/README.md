# EQ560 - 7-Band EQ Audio Plugin
## c176 Studio

### Installation

#### Windows
1. Extract the ZIP
2. Right-click `Install.bat` °˙ Run as Administrator
3. Restart your DAW

#### macOS (Apple Silicon - M1/M2/M3)
```bash
cd ~/Downloads/EQ560_macOS_ARM
chmod +x install.sh
sudo ./install.sh
```

#### macOS (Intel)
```bash
cd ~/Downloads/EQ560_macOS_Intel
chmod +x install.sh
sudo ./install.sh
```

### Features
- HP/LP Filters (10Hz - 21.5kHz)
- Hi/Lo Shelving EQ
- 3x Peak EQ with Q control
- Dual Pultec EQP-1A style sections
- Digital / Passive / Active modes
- Real-time EQ curve display
- Vintage rack-style UI

### Formats
- VST3 (Windows, macOS ARM, macOS Intel)
- AAX (Windows - requires PACE signing for Pro Tools)
